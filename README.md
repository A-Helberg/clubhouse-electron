# Clubhouse packaged as an electron app
### Why?
Because I can never find it in my open tabs

# Build

* `yarn make`

# Overview
This project is made with electron-forge and really only opens the clubhouse website
PRs welcome


# Deps

On mac you will need `brew install fakeroot dpk rpm`
To build for linux platforms run `npx electron-forge make --platform=linux --arch=ia32,x64`
